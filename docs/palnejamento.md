# Planejamento

## Objetivo
Criar um formulário HTML5 com validação de campos usando JavaScript puro.

## Funcionalidades
- Nome obrigatório.
- Email obrigatório e com formato válido.
- Senha obrigatória com no mínimo 6 caracteres.
